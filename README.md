## 🧬 Core Features (MVP)

1. **AI Resume X-Ray** — Upload resume + target JD → structured feedback with scores
2. **Semantic Job Matching** — Meaning-based matching with plain-language explanations
3. **Before/After Diff** — Side-by-side resume comparison with annotated changes
